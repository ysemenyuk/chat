Simple chat demo project

#### Built With

- react
- react-router-dom
- @reduxjs/toolkit
- socket.io
- react-bootstrap
- formik
- axios

#### Deploy

https://ysemenyuk-simple-chat.herokuapp.com/

#### Requirements

* [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)

#### Install

```sh
$ make install
```

#### Run

```sh
$ make start
# open http://localhost:5000
```
